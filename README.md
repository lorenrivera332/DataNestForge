# DataNestForge - Structured Data Builder

DataNestForge creates realistic datasets by nesting field groups and chaining field generators.

## Features
- Nested field groups  
- JSON, CSV, SQL export  
- CLI generator tool

## Usage
```bash
git clone https://github.com/your-username/22_DataNestForge.git
cd 22_DataNestForge
python generate.py
